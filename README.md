# School Management System

## 📌 Description
This project is a School Management System built using microservices architecture.

## 🚀 Tech Stack
- Backend: Node.js, Express
- Frontend: React
- Database: MySQL, MongoDB
- Authentication: JWT

## 🔐 Features Implemented
- User registration (default role: Student)
- User login with JWT authentication
- Role-based access control (Admin, Teacher, Student, Parent)
- Super Admin privileges
- Admin can create users (Teacher, Parent, Admin)
- Parent-Student relationship (many-to-many)

## 🏗️ Architecture
- Microservices:
  - Auth Service
  - API Gateway (planned)
  - Other services (in progress)
